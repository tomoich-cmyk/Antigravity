## パイロット後 ロードマップ

### 優先度 A (次のマイルストーン)
**Chrome 拡張のトリガールールを Android に持ってくる**
- 各銘柄の閾値・売買方向を Android に同期
- 現在値がトリガー圏内かを Android 上で判定・表示
- 「監視」→「判断支援」へのステップアップ

### 優先度 B
**AB / インベスコの実基準価額取得**
- 投信は市場 API 対象外のため、取得元の選定が必要
  - 運用会社 API / スクレイピング / 手動入力 のいずれか
- それまでは mock 値のまま運用

### 優先度 C
**サマリーを「状況」から「推奨行動」へ**
- トリガールール移植完了後に着手
- 「GMO-PG が閾値に接近」「AB は様子見」等の文言生成
