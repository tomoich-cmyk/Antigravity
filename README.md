# Antigravity (アンチグラビティ)

Antigravity は、個人投資家が株式市場の動向を冷静に分析し、投資判断を支援するためのツール群です。
Chrome 拡張機能による直感的なインターフェースと、J-Quants API 等を活用した市場データ取得サーバーで構成されています。

## プロジェクト構成

このリポジトリは、以下の 2 つの主要なコンポーネントで構成されています。

- **antigravity-ext**: Chrome 拡張機能 (フロントエンド)
  - React + Vite + TypeScript で構築。
  - ポートフォリオの管理、取引ログの記録、投資判断（スコアリング）の表示を行います。
- **antigravity-server**: データ取得・解析サーバー (バックエンド)
  - Node.js + TypeScript で構築。
  - 市場データ（株価、為替、指数など）の定期的な取得と加工を担当します。

## 主な機能

- **投資判断エンジン**: 市場状況（USD/JPY, SPY, VT 等）に基づいた、独自のスコアリングによる売買推奨（Strong Buy / Buy / Wait / Stop）。
- **ポートフォリオ管理**: 現金残高、保有資産の推移、実現損益の可視化。
- **自動データ更新**: 定期的な市場データ取得による、常に最新の状況に基づいた分析。
- **ローカルファースト**: プライバシーを重視し、重要なデータはローカル環境を中心に管理。

## 開発の哲学

「重力（市場の暴落やパニック）に抗い、冷静な判断を下す」ことを目的としています。過度な装飾を排し、実用性とデータに基づいた意思決定をサポートします。
